## 🎬 User Journey

```
1. DISCOVER
   Landing on homepage → Hero catches attention
   Sees featured products & countdown
   Feels luxury and exclusivity

2. EXPLORE
   Clicks "View Details" on product
   Sees high-quality image
   Reviews fragrance notes
   Checks social proof (viewers, stock)

3. CHOOSE
   Selects volume (10/20/30/50ml)
   Price updates automatically
   Clicks "Add to Reserve"
   Sees success animation

4. REVIEW
   Goes to cart
   Reviews items and pricing
   Adjusts quantities if needed
   Sees "Complimentary Shipping"

5. PURCHASE
   Clicks "Proceed to Checkout"
   Fills contact info
   Enters shipping address
   (Optionally fills payment info)
   Clicks "Place Order"

6. CONFIRM
   Sees success page
   Appreciative message sent
   Option to return home
   Cart clears for next visit
```

## 🔄 How to Customize

### **Change Product Images**
Edit: `src/data/perfumes.ts`
```typescript
imageUrl: "https://your-image-url.com/perfume.jpg"
```

### **Update Product Names**
Edit: `src/data/perfumes.ts`
```typescript
name: "Your New Name"
```

### **Modify Pricing**
Edit: `src/data/perfumes.ts`
```typescript
export const VOLUME_PRICES = [
  { ml: 10, price: 8 },  // Changed price
]
```

### **Change Colors**
Edit: `src/index.css`
```css
--primary: 46 57% 52%;  /* Your color */
```
